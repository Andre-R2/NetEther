# NetEther

NetEther is a personal documentation project where I organize technical notes, practical configurations, diagrams and reference material. 

## Documentation

### Networking
Fundamentals, switching, VLANs, STP, and enterprise topology architectures.

### Linux
Ubuntu Server, Bash, filesystem permissions, systemd, SSH, Nginx, DNS/DHCP, Samba.

### Cybersecurity
Firewalls, VPNs, IDS/IPS, and system hardening practices.

### Hands-on Labs
Practical labs in Packet Tracer, GNS3, EVE-NG, VirtualBox, and VMware.
